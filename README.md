# Rectangle Calculator CLI (Node.js)

A simple Command Line Interface (CLI) application built with Node.js to calculate
the **area** and **perimeter** of a rectangle.  
This project also saves calculation history to both **TXT** and **JSON** files.

- Async/Await based CLI
- Exit confirmation for better user experience
